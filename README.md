# Mobile Flight Itinerary App

This project is part of the **Skyscanner Software Engineering Job Simulation** on Forage. The goal of this task was to develop a mobile flight itinerary display using Android Studio and Skyscanner’s open-source design system, **Backpack**.

## ✈️ Project Overview
The application displays a user's flight itinerary, including flight numbers, departure details, and arrival information, all styled using Backpack components to ensure a consistent and visually pleasing UI.

## 🛠️ Features & Implementation
- **Backpack Integration:** Utilized the `net.skyscanner.backpack:backpack-android` library for UI components.
- **Flight Information Card:** Displays the flight number.
- **Departure Card:** Shows a three-digit airport code and departure time.
- **Arrival Card:** Shows a three-digit airport code and arrival time.
- **Custom Styling:** Implemented `BpkCardView` with large corner styles and `BpkText` with heading styles for better hierarchy.

## 🚀 Tech Stack
* **Language:** Kotlin
* **IDE:** Android Studio
* **UI Framework:** Backpack (Android)
* **Minimum SDK:** API 33 (Android Tiramisu)
